python字典

字典的定义

字典是python中唯一的映射类型(哈希表)。
字典对象是可变的，但字典的键必须使用不可变对象，一个字典可以使用不同类型的键值。字典的输出是无序的。

字典的方法

  • keys()
    返回列表，内容为字典的key。

  • values()
    返回列表，内容为字典中的value。

  • items()
    返回列表，内容为字典的键值对。
