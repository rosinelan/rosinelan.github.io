python流程控制之if

if

格式

缩进使用4个空格，不建议使用TAB键。

实例

键盘输入数字，>90输出A,>80输出B，>=70输出C，<70输出D。

逻辑值包含2个量。

  • True：表示非空的量，所有非零数。
  • False：表示0，none，空的量等。
