python访问redis(二)

关于常用nosql数据库介绍及redis安装配置的介绍，点击python访问redis查看。
前面介绍了redis常用的数据类型，包括String、List、Hash、Set和Sorted Set。下面针对不同数据类型的操作进行介绍。

String

  • set(name, value)
    设置值，默认不存在则创建，存在则修改

  • get(name)
    获取值

  • mset(args, *kwargs)
    批量设置值，传入参数为类型为dict

  • mget(keys, *args)
    批量获取key的值，传入参数类型为tuple

或者使用下面传递参数

  • append(key, value)
    在key对应的value后面追加内容

List

在redis中，list类型的存储按照一个name对应一个list来存储。

  • lpush(name, *values)
    左边添加

或者可以如下

  • rpush(name, *values)
    右边添加

  • linsert(name, where, refvalue, value)
    根据where参数，在列表name的值refvalue前面或者后面插入值value，成功则返回值为新的列表的长度，否则返回-1。
    where取值为before或者after。

  • lpop(name)
    删除列表name中的第一个元素，返回值为删除的元素

  • lrange(name, start, end)
    通过分片返回list位于start和end范围的值

  • lset(name, index, value)
    修改list中指定位置的值

  • lrem(name, value, num)
    删除列表name中最先出现的num个值为value的元素。
    num默认值为0，表示删除全部值为value的元素；
    num>0表示删除值为value的元素时从头到尾匹配；
    num>0表示删除值为value的元素时从尾到头匹配。

Set

Set集合就是不允许重复元素的列表。

  • sadd(name, *values)
    集合name中添加值。
  • scard(name)
    返回集合name中元素的数量。

  • smembers(name)
    返回集合的所有成员。

  • spop(name)
    随机删除并返回集合name中的一个元素。

  • srem(name, *values)
    从集合name中删除指定的值。

  • sunion(keys, *args)
    返回多个集合的并集

  • sinter()
    返回多个集合的交集

Zset

Zset也就是sorted Set，有序集合。在集合的基础上元素的排序根据另外一个值进行比较。所以有序集合每个元素有2个值，即值和分数，分数专门用来做排序。

  • zadd(name, args, *kwargs)
    集合添加元素，下面两种写法是等价的。

  • zcard(name)
    有序集合中名称为name的元素个数。

  • zrange(name, start, end)
    返回有序集合name中位于start和end之间的元素，默认为升序排列。

  • zrem(name, *values)
    删除有序集合name中的值。

Hash

redis中的Hash在内存中类似于一个name对应一个dict来存储。

  • hset(name, key, value)
    名称为name的hash中，将key的值设置为value。
    若成功添加则返回1，否则返回0.

  • hget(name, key)
    在名称的name的hash中，返回key的对应值。

  • hmset(name, mapping)
    名称为name的hash中，根据mapping字典内容添加对应的key和value。

  • hmget(name, keys, *args)
    名称为name的hash中获取多个keys的值。

  • hkeys(name)
    列表形式返回名称为name的hash中的keys。

  • hvals()
    列表形式返回名称为name的hash中的values。

  • hexists(name, key)
    名称为name的hash中是否存在key。返回值为布尔值。

  • hdel(name, keys)
    删除名称为name的hash中的keys。

  • hlen(name)
    返回名称为name的hash中元素的个数。

其它常用操作

下面的操作适用于上述所有类型

  • keys(pattern=’*’)
    返回匹配pattern的key。

  • delete(names)
    删除    name指定的一个或多个keys。

  • exists(name)
    判断key为name是否存在。返回值为布尔值。

  • rename(src, dst)
    修改key的名称，修改成功返回True，否则返回错误。

  • expire(name, time)
    设置key的过期时间。

  • type()
    返回key所存储值的类型。

  • move(name, db)
    将key为name的元素迁移至db。

  • flushall()
    删除所有的key。
